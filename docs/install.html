Installation

As much as I would like to make Spinna a turnkey project, some extra installation steps are required.

Requirements

In addition to itself, Spinna requires the following.

  • rDMD or other D compiler
  • Sass
  • Dub packages jaypha-base, jaypha-inet, dyaml and their dependencies.

For FCGI based websites:

  • FCGI system library
  • Dub package fcgi-loop

For Vibe-d base websites:

  • Dub package vibe-d

For console executables:

  • Dub package backtrace

MakeRouter

To make use of Spinna's router, you will need to compile the makerouter utility and install it.

The main source file is in src/makerouter.d

A makefile exists for Linux. To use it run make bin and then, sudo make install.
