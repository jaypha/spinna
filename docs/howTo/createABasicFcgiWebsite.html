How To Create a Basic FCGI Website

It is assumed that you already know how to:

  • Construct a project in your chosen environment.
  • Setup your web server for use with FCGI.

Do the following

  • Install Spinna and fcgi-loop
  • Create a project.
  • Copy config/general.d into your project.
  • Create a module source file.
  • Import jaypha.spinna.main.fcgi
  • Create functions for error handling, pre processing and post processing (if desired)
  • Inside a shared static this(), assign function handlers to fcgiRequestProcessor.
  • Create modules and functions to process the requests.
  • You will need to create a YAML router definition file. (see Router Definition and Usage)
  • Compile the project.
  • Put the generated program into your website.

A Sample exists in samples/basic to serve as a guide.

Note

The fcgi loop interface works by creating a number of threads and reusing those threads for new requests. Any global variables set during a process will carry on to the next request. Therefore it is important to ensure that any global variables used are reset between requests. Any global variables used by the framework are automatically reset by the framework.
