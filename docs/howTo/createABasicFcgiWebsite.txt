Create a basic website.

- This doc assumes you know how to construct a project in your chosen environment.
- This doc assumes you know how to construct an fcgi website for your chosen web
server.

- Install Spinna.
- Create a project.
- You will need to create a router yaml file.
- Create a module source file.
- Import jaypha.spinna.main.fcgi;
- Create functions for error handling, pre processing and post porcessing (if desired)
- Inside a shared static this(), assign function handlers to fcgiSpinnaRequestProcessor.
- Compile the project.
- Put the generated program into your website.
- sample exists in samples/basic to serve as a guide.
