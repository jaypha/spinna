Installing Spinna

- This doc assumes you know how to construct a project in your chosen environment.

- Spinna requires the following
  - rDMD
  - Sass
  - fcgi
  - Dub packages fcgi-loop, backtrace, dyaml and their dependencies.
- Download/unzip/clone project into a suitable directory.
- You will need to compile and install makerouter into a suitable executable directory.
